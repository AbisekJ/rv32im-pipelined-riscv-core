# RV32I+M 5-Stage Pipelined RISC-V Core

## Overview
32-bit pipelined RISC-V processor designed in Verilog and implemented on Arty A7 FPGA.

## Features
- 5 stage pipeline
- Hazard detection
- Data forwarding
- RV32IM support

## Tools Used
- Verilog
- Vivado
- Arty A7

## Folder Structure
rtl/ : source files  
tb/ : testbench  

## Results
Validated at 50 MHz.
